# Imported Image Assets

อัปเดต: 28 พ.ค. 2026

นำผลลัพธ์จาก Codex ตัวรวบรวมข้อมูลเข้ามาเก็บใน repo แล้ว

## ตำแหน่งไฟล์

- JSON ดิบที่ได้มา: `data-imports/acnh_curated_post_images.raw.json`
- รูปที่ดาวน์โหลดแล้ว: `public/content-images/curated-post-images/`
- ข้อมูลที่เกลาแล้วสำหรับใช้ในเว็บ: `src/data/postImageAssets.ts`

## สรุปผลนำเข้า

- จำนวน guide ที่มีรูป: 15
- จำนวนรูปที่ดาวน์โหลดสำเร็จ: 50
- ขนาดโฟลเดอร์รูป: ประมาณ 4.7 MB
- ไม่มีรายการ blocked จากผลลัพธ์ล่าสุด

## แนวทางใช้งานในเว็บ

อย่าใช้ `postText` จาก JSON ดิบโดยตรง เพราะยังมีข้อมูลจาก Facebook ปนอยู่ เช่น ชื่อกลุ่ม ชื่อผู้โพสต์ reaction count และ comment count

ให้ใช้ `src/data/postImageAssets.ts` แทน โดยข้อมูลถูกเกลาเป็น:

- `summary`: คำอธิบายสำหรับ card/list
- `sourceExcerpt`: ข้อความต้นทางที่เรียบเรียงให้อ่านเข้ากับสไตล์เว็บ
- `images`: path รูปใน `public/` ที่เว็บเรียกได้ตรง ๆ

## หมายเหตุสำคัญ

- รูปถูกเก็บไว้เพื่อใช้ประกอบโพสต์ที่เราจะนำเข้าเว็บ ไม่ใช่ผล OCR
- ยังควรตรวจสิทธิ์การใช้ภาพก่อนเผยแพร่สาธารณะ
- หากรูปมีข้อมูลส่วนตัวหรือหน้าคน ควรให้แอดมิน review เพิ่มก่อนนำขึ้น production
